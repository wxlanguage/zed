# WX for Zed

Language support for [WX](https://github.com/wxlanguage/wx), a Rust-inspired language that compiles directly to WebAssembly, in the [Zed](https://zed.dev) editor.

Not yet published — this repo is a placeholder while the extension is being built. See [wxlanguage/vscode](https://github.com/wxlanguage/vscode) for the existing VS Code extension, which this one is expected to mirror in functionality (diagnostics, hover, completions, formatting, etc. via `wx lsp`).
